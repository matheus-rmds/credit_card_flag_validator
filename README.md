# Validador de Bandeiras de Cartão de Crédito

Validador simples em Python que identifica tipos de cartão pelo prefixo
do número usando padrões regex.

O que faz
- Detecta tipos comuns de cartão: Visa, MasterCard, American Express,
  Discover, Hipercard e Elo com base no prefixo do número do cartão.

Como executar
1. Abra um terminal na pasta raiz do projeto.
2. Execute o script de testes (o script está em `src`):

```bash
cd src
python app.py
```

Você verá uma tabela curta com os resultados dos testes e um resumo dos
casos aprovados/falhados. Pode ser alterado ou adicionado para mais testes.
